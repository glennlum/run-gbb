About

☀️ Sunday Morning Run at Gardens by the Bay!

  • Membership : Open (NDU 34TH)
  • Commitment : Voluntary
  • Activity : 5K Run & Chat
  • Frequency : Every first Sunday of the month.
  • Time : 0830H-1000H
  • Place : Meet outside GBB McDonald’s.
  • Barang : Lockers are available for bag storage.
  • Restrooms : Present at start and finish point.
  • Parking : Basement car park at main entrance.
  • MRT : Bayfront CE1/DT16
  • Cancellation : If in the event the first Sunday slot is cancelled, the third Sunday (reserve) slot will be activated.
